


1、想要实现异步加载订单列表页面中，每一个订单中的订单详情；

	a、首先要在该订单详情框中设置一个按钮，并将该按钮设置value=“订单详情”，id=“but+订单编号”，接着为该按钮设置一个点击事件，当点击该按钮的时候，触发事件

	b、该点击事件主要就是实现异步加载该订单的订单项：触发该事件后，首先要获得订单详情按钮的对象、订单详情按钮所在的div对象；接着使用if判断该按钮的value的值是否为
	   “订单详情”；若是则创建异步对象，并设置监听，即回调函数；接着打开链接跳转到相应的Action中，并将该订单的oid传到action中，最后在Action中通过该oid将该oid
	   对应的订单对象查出来，然后通过该订单对象找到该订单对象对应的订单项。

	c、将查询所得的订单项组成的list存储在值栈中，然后通过struts的配置文件跳转到orderItem.jsp页面，

	d、然后在异步加载代码中将按钮的value值改为“关闭”，并且触发回调函数，将orderItem.jsp页面的内容设置到订单详情的方框中。

	e、当再次点击该按钮的时候，也会进行判断，因为第一次点击时，该按钮的value值已经改为“关闭”，那么就是不能进到if中，也就是只能进到else中，将div1.innerHTML 设	   	   置为 "";即 div1.innerHTML = "";接着又将按钮的value值改为“订单详情”，即but.value="订单详情";那么下次点击时又能进到if中，能看到详情。

			//接收onclick方法传过来的参数oid
			function showDetail(oid){
				//获得按钮对象
				var but = document.getElementById("but"+oid);
				//获得div对象
				var div1 = document.getElementById("div"+oid);
				
				//判断按钮对象的值是否为“订单详情”，若是，则创建异步对象，执行回调函数，并将Action中返回的内容显示在orderItem.jsp页面中，
				//然后将该页面嵌套订单详情所在的表格中
				
				//倘若要实现点击订单详情按钮就将订单的详情显示出来，并且该订单详情的按钮要变为关闭按钮；
				//那么就要在显示完订单详情后，将“订单详情”按钮的值改为“关闭”，点击该关闭按钮的时候又会触发这一个异步加载，，那么再一次进行判断时
				//，该按钮的值不是订单详情，就不会进到if中，只能进到else中，在else中将订单详情区域的div的内容设置为 “”，那么就等于关闭了该详情。

				if(but.value == "订单详情"){

					// 1.创建异步对象
					var xhr = createXmlHttp();

					// 2.设置监听（这里的function是一个回调函数）
					xhr.onreadystatechange = function(){

						if(xhr.readyState == 4){
							if(xhr.status == 200){
								
								//将返回的内容显示到该div
								div1.innerHTML = xhr.responseText;
							}
						}
					}

			// 3.打开连接（请求方式、请求路径、true）
			xhr.open("GET","${pageContext.request.contextPath}/adminOrder_findOrderItem.action?oid="+oid+"&time="+new Date().getTime(),true);

					// 4.发送（发送请求）
					xhr.send(null);
					but.value = "关闭";
				}else{
					div1.innerHTML = "";
					but.value="订单详情";
				}
				
			}
			function createXmlHttp(){
				   var xmlHttp;
				   try{ // Firefox, Opera 8.0+, Safari
				        xmlHttp=new XMLHttpRequest();
				    }
				    catch (e){
					   try{// Internet Explorer
					         xmlHttp=new ActiveXObject("Msxml2.XMLHTTP");
					      }
					    catch (e){
					      try{
					         xmlHttp=new ActiveXObject("Microsoft.XMLHTTP");
					      }
					      catch (e){}
					      }
				    }

					return xmlHttp;
				 }
		</script>
